Tentang Manajemen User / FAQ

Apa perbedaan role Admin dan Karyawan?

Admin memiliki akses penuh ke seluruh fitur dashboard termasuk manajemen user dan pengaturan sistem. Karyawan hanya dapat melihat dan mengelola data transaksi sesuai dengan departemen mereka.

Bagaimana cara mereset password user?

Klik tombol 'Detail' pada user yang bersangkutan, lalu pilih 'Reset Password' di bagian bawah panel detail. Tautan reset akan dikirimkan ke email user.
